Trending Movies

Picture by J.Pihlgren
Description

This is a personal project from when I did the The Web Developer Bootcamp on Udemy. The app scrapes 50 trending movies from IMDB using imdbScrapper and stores title, rating, synopsis, year and an image to a mongoDB database. This so that the app doesn't query imdbScrapper everytime the page reloads. It isn't finished but works good enough so that I can see what the trending movies are.

It uses node.js, express, mongo, mongoose, imdbScrapper, semanticUI and ejs to name a few.

The project
What
A web app that scrapes trending movies from IMDB.
Why
I don't have time to browse imdb for movies I want to watch.
Aim
Provide myself with a quick overview of the latest movies.
